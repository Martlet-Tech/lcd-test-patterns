About LCD Test Patterns

LCD Test Patterns is a web-based tool for generating LCD display test images. It is a port of the open-source C# Windows Forms application TestPictrueGenerator.

Features:

  • Standard test patterns: pure colors, gray gradients, color bars
  • LCD characterization: crosstalk, flicker, pixel response
  • Scale/ruler patterns for pixel mapping
  • AllRGB images with unique colors per pixel
  • DPI timing calculator for display driver configuration
  • DSI bitrate calculation
  • Cut panel mode for partial-screen displays

Built for LCD debug engineers, display driver developers, and hardware validation teams.
